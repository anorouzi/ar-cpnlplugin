# ar-cpnlplugin
ArvanCloud cPanel Plugin

# Brief
ArvanCloud cPanel Plugin to add and config domain in CDN panel directly from cPanel

## Input
User API Token available in arvancloud panel

## Capabalities
* Register Domain and get NSs
* Config Cache/SSL
* Add DNS Records
